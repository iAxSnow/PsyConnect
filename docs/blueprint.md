# **App Name**: Profe UDP Connect

## Core Features:

- Autenticación Segura: Autenticación segura de usuarios con verificación de correo electrónico institucional y carga de foto de perfil.
- Búsqueda de Cursos y Tutores: Funcionalidad de búsqueda avanzada con sugerencias de autocompletado para cursos y tutores.
- Visualización de Perfiles de Tutores: Perfiles de tutores que muestran foto, nombre, calificaciones, cursos destacados y tarifas por hora.
- Flujo de Solicitud de Tutor: Flujo de trabajo para que los estudiantes presenten una solicitud para ser un tutor verificado. Incluye la carga de certificados, la selección de cursos y el establecimiento de precios.
- Gestión de Perfil de Usuario: Página de perfil personal con opción para editar el perfil de usuario y listar las sesiones de tutoría programadas e históricas.
- Sistema de Calificación y Revisión: Función de calificación posterior a la sesión utilizando calificaciones de estrellas y comentarios, activada al marcar una sesión de tutoría como completada.
- Recomendaciones de Tutores con IA: Herramienta "Sugerir un tutor" impulsada por IA: Basado en los cursos en los que está inscrito, utiliza IA generativa para sugerir tutores que otros estudiantes como usted hayan utilizado en el pasado.

## Style Guidelines:

- Color primario: Azul UDP (#004395) para acciones principales y marca. El tono representa confianza y profesionalismo, al tiempo que alude a la identidad de la escuela.
- Color de fondo: Gris claro (#F5F5F5) para una apariencia limpia y ordenada, que promueve la facilidad de uso.
- Color de acento: Naranja UDP (#E37222) para resaltar alertas y elementos interactivos clave. Esto llama la atención del usuario sobre elementos importantes de la interfaz de usuario sin sobrecargar el esquema de color azul-gris.
- Fuente del cuerpo y del encabezado: 'Inter', una fuente sans-serif, para una alta legibilidad y una interfaz moderna y limpia.
- Iconos vectoriales nítidos y claros para facilitar el reconocimiento de acciones y categorías.
- Diseño minimalista con amplio espacio en blanco para mejorar el escaneo de contenido y la usabilidad general.
- Animaciones de transición sutiles y retroalimentación para las interacciones del usuario para mejorar el compromiso.
